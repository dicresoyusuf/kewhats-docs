## Lupa Password?

1. Klik **"Forgot Password"** di halaman login
2. Masukkan email terdaftar
3. Cek inbox untuk link reset
4. Klik link, buat password baru
5. Login dengan password baru

---

## Troubleshooting

| Masalah | Solusi |
|---------|--------|
| "Email sudah terdaftar" | Gunakan email lain atau reset password |
| "Password salah" | Pastikan caps lock off, gunakan forgot password |
| Tidak bisa login | Pastikan email sudah diverifikasi |
| Link verifikasi expired | Request kirim ulang dari halaman login |
