Cara membuat dan mengelola knowledge base.

## Langkah Pembuatan

### Step 1: Buka Smart Knowledge
Klik **Smart Knowledge** di sidebar.

### Step 2: Create New
Klik **"Create Knowledge Base"**.

### Step 3: Isi Detail

| Field | Contoh |
|-------|--------|
| Name | FAQ Produk |
| Description | Info produk dan harga |
| Content | Isi konten... |

### Step 4: Format Konten

```markdown
## FAQ Pengiriman

Q: Berapa ongkir ke Jakarta?
A: Ongkir ke Jakarta mulai Rp15.000.

Q: Berapa lama pengiriman?
A: Reguler 2-3 hari, Express 1 hari.
```

### Step 5: Save & Process
1. Klik **Save**
2. Klik **Process** untuk embedding
3. Tunggu status **"ready"**
