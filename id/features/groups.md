## Kirim Pesan ke Grup

1. Pilih grup
2. Klik **"Send Message"**
3. Ketik pesan
4. Send!

---

## Export Members

1. Pilih grup
2. Tab **"Members"**
3. Klik **"Export"**
4. Download CSV

Gunakan untuk broadcast atau import ke contacts.

---

## Limitasi

::: warning Catatan
- Tidak bisa create grup dari sistem
- Tidak bisa add/remove member
- Rate limit grup lebih ketat
:::

---

## Best Practices

::: tip Tips
1. Jangan spam grup (max 1-2 pesan/hari)
2. Pesan yang relevan dengan grup
3. Respect peraturan grup
:::
