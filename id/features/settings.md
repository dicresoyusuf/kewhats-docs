# ⚙️ Settings

Konfigurasi akun dan preferensi.

## Sections

| Section | Deskripsi |
|---------|-----------|
| General | Timezone, language, theme |
| Account | Profil & password |
| API | API key management |
| AI Configuration | Setup AI keys |
| Notifications | Preferensi notifikasi |

---

## Account

### Update Profile
Settings → Account → Edit nama, email, foto

### Change Password
Settings → Security → Input password baru

---

## API Key

1. Settings → API
2. View key (masked)
3. Klik "Show" untuk reveal
4. "Regenerate" untuk key baru

::: warning
Key lama langsung invalid setelah regenerate!
:::
