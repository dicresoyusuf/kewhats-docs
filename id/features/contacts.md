## Menambah Kontak

### Manual:
1. Klik **"Add Contact"**
2. Isi data:
   - Name
   - Phone (format: 6281xxx)
   - Tags
   - Notes
3. Save

### Import CSV:
1. Siapkan file CSV:
   ```csv
   name,phone,email,tags
   John Doe,6281234567890,john@email.com,"customer,jakarta"
   ```
2. Klik **"Import"**
3. Upload file
4. Preview & confirm
