Saran balasan otomatis yang diaktifkan oleh AI.

## Overview

Smart Reply menganalisis pesan masuk dan menyarankan 3 balasan yang relevan dan natural.

---

## Cara Kerja

1. **Pesan masuk** dari customer
2. **AI analisis** konteks dan intent
3. **Generate** 3 saran balasan
4. **Pilih** yang sesuai, edit jika perlu
5. **Kirim**!

---

## Lokasi

Di halaman **Inbox**, scroll ke bawah input message. Anda akan melihat panel Smart Reply dengan 3 saran.

---

## Knowledge Mode

Toggle icon ðŸ§  (Brain) untuk mengaktifkan:

| Mode | Behavior |
|------|----------|
| **OFF** | Saran generik berdasarkan context |
| **ON** | Saran berdasarkan Knowledge Base |

Saat ON, badge "Knowledge" muncul dan saran akan menggunakan informasi dari knowledge base Anda.

---

## Indikator

- **Brain icon biru**: Knowledge mode aktif
- **Badge "Knowledge"**: Saran dari KB
- **Purple border**: Saran yang berasal dari knowledge

---

## Tips

::: tip Best Practices
1. Review saran sebelum kirim - AI adalah asisten, bukan pengganti
2. Edit untuk personalisasi
3. Gunakan Knowledge mode untuk FAQ/product info
4. Disable untuk percakapan casual
:::
