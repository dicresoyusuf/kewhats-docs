## Send Message to Group

1. Select group
2. Click **"Send Message"**
3. Type message
4. Send!

---

## Export Members

1. Select group
2. Tab **"Members"**
3. Click **"Export"**
4. Download CSV

Use for broadcasts or import to contacts.

---

## Limitations

::: warning Note
- Cannot create groups from system
- Cannot add/remove members
- Group rate limits are stricter
:::

---

## Best Practices

::: tip Tips
1. Don't spam groups (max 1-2 messages/day)
2. Send relevant messages to the group
3. Respect group rules
