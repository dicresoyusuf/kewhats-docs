| Section | Description |
|---------|-------------|
| General | Timezone, language, theme |
| Account | Profile & password |

| AI Configuration | AI keys setup |
| Notifications | Notification preferences |

---

## Account

### Update Profile
Settings → Account → Edit name, email, photo

### Change Password
Settings → Security → Enter new password

---

## API Key

1. Settings → API
2. View key (masked)
3. Click "Show" to reveal
4. "Regenerate" for new key

::: warning
Old key becomes invalid immediately after regenerate!
:::
