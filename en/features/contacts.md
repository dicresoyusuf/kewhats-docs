## Adding Contacts

### Manual:
1. Click **"Add Contact"**
2. Fill data:
   - Name
   - Phone (format: 1234567890)
   - Tags
   - Notes
3. Save

### Import CSV:
1. Prepare CSV file:
   ```csv
   name,phone,email,tags
   John Doe,1234567890,john@email.com,"customer,newyork"
   ```
2. Click **"Import"**
3. Upload file
4. Preview & confirm
