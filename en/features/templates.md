## Using Templates

### In Inbox:
1. Type `/` in message input
2. Select template from list
3. Template auto-inserts
4. Edit if needed, then send

---

## Best Practices

::: tip Tips
1. Create templates for frequently asked questions
2. Use emoji for visual appeal
3. Keep it short - max 3-4 lines
4. Include call-to-action
:::
