## Google Sheets

### Features:
- Sync contacts from/to Sheets
- Auto-update on new data
- Bidirectional sync

### Setup:
1. Click **"Connect"**
2. Login to Google
3. Authorize
4. Select spreadsheet
5. Map columns
6. Enable

---

## Telegram

### Features:
- Forward incoming messages to bot/group
- Important notifications
- Reply from Telegram

### Setup:
1. Create bot via @BotFather
2. Get token
3. Paste in KeWhats
4. Set chat ID

---

## Discord / Slack

### Features:
- Forward messages to channel
- Webhook notifications

### Setup:
1. Create incoming webhook
2. Copy URL
3. Paste in KeWhats
4. Enable
