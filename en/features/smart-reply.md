AI-powered automatic reply suggestions.

## Overview

Smart Reply analyzes incoming messages and suggests 3 relevant and natural replies.

---

## How It Works

1. **Message arrives** from customer
2. **AI analyzes** context and intent
3. **Generates** 3 reply suggestions
4. **Select** the appropriate one, edit if needed
5. **Send**!

---

## Location

On the **Inbox** page, scroll below the message input. You'll see the Smart Reply panel with 3 suggestions.

---

## Knowledge Mode

Toggle the ðŸ§  (Brain) icon to enable:

| Mode | Behavior |
|------|----------|
| **OFF** | Generic suggestions based on context |
| **ON** | Suggestions based on Knowledge Base |

When ON, a "Knowledge" badge appears and suggestions use information from your knowledge base.

---

## Indicators

- **Blue Brain icon**: Knowledge mode active
- **"Knowledge" badge**: Suggestion from KB
- **Purple border**: Suggestions from knowledge

---

## Tips

::: tip Best Practices
1. Review suggestions before sending - AI is an assistant, not a replacement
2. Edit for personalization
3. Use Knowledge mode for FAQ/product info
4. Disable for casual conversations
:::
